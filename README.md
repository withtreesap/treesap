# Treesap

> **Warning**
> This is a work in progress. If you're reading this you are likely here too early. The API is not stable and may change at any time.

This is the source code for Treesap, a modern frontend framework for building insanely fast websites.

It is built using Lit, Deno, and Web Components. Every page is server side rendered by default and then web components are hydrated on the client as needed.

It takes inspiration from a number of amazing projects including but not limited to:

- Fresh by Deno
- Astro
- Enhance
- WebC from 11ty
- NextJS
